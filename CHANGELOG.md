# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-draft] - 2026-03-16

### Added
- Initial project setup in Zig.
- HTML parser and DOM tree.
- CSS tokenizer and style resolver.
- Layout engine (Block, Inline, Flexbox).
- GPU-accelerated Metal renderer.
- JavaScriptCore integration.
- Networking with `NSURLSession`.
- Basic UI with URL bar and scrolling.
- CI/CD foundation.
