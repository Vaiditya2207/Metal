Metal Browser Engine

A macOS-native browser engine built with Zig and Apple Metal. This page demonstrates HTML parsing, CSS styling, block and inline layout, flex containers, and word wrapping.

Rendering Pipeline

The engine reads an HTML file from disk, tokenizes it into a DOM tree, extracts and parses CSS from style elements, resolves computed styles using cascade and specificity rules, performs block and inline layout with word wrapping, builds a display list of drawing commands, and rasterizes everything through a batched Metal pipeline with MSDF text rendering.

DOM

Full HTML5 tokenizer with entity decoding, void element handling, and tree construction. Supports attribute and tree mutation APIs.

CSS

Tokenizer, parser, selector matching with specificity, cascade resolution, and a user-agent stylesheet. Supports class, ID, and type selectors.

Layout

Block formatting context, inline word wrapping, flexbox with grow and shrink, auto margins, margin collapsing, box-sizing, and min-max constraints.

Phase 6: File loading, style extraction, word wrapping, glyph metrics bridge, and JavaScript execution via JavaScriptCore.
